# Flying Fingers ✨

A Chrome extension that helps you level up your typing. Track your real-time words-per-minute across the web, earn ranks as you improve, and sharpen your speed with focused practice sessions.

> **Status:** Early development. Planning complete, scaffolding next.

## Features (Planned)

- **Real-time WPM tracking** — measures your typing speed on pages you actually use, not contrived tests
- **Rank progression** — climb 10 tiers from 🐾 Hunt & Peck to ✨ Flying Fingers as your skills grow
- **Practice mode** — pace yourself against a target WPM with built-in passages to build muscle memory
- **Custom passages** — load your own text for targeted practice (enable in Advanced Settings)
- **Stats dashboard** — WPM-over-time chart, session history, accuracy, streak tracking
- **Dark mode** — because your eyes deserve a break
- **100% local** — no accounts, no servers, no telemetry; your typing data never leaves your browser

## Supported Platforms

Flying Fingers runs wherever Chrome or a Chromium-based browser runs:

- macOS
- Windows
- ChromeOS (Chromebook)
- Linux

## Tech Stack

- **Vite** + **TypeScript** + **Preact** — fast builds, type-safe engine code, tiny runtime
- **Dexie** — local IndexedDB storage for session history
- **Manifest V3** — modern Chrome extension APIs
